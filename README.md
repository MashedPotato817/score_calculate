# 斗地主计分器

基于 Python tkinter 的斗地主计分图形界面程序，支持大局/小局两层结构、文件持久化、历史管理。

## 环境要求

- Python 3.7+
- tkinter（Python 标准库，无需额外安装）

## 运行

```bash
python main.py
```

或双击 `main.py` 文件直接运行。

## 功能

- **三人记分**：支持修改玩家姓名，每个大局内分数累计，实时显示
- **番数调节**：+1 番 / +2 番 / 重置番数
- **地主标记**：点击切换地主，同一时间只能有一个地主，按钮显示 👑 标识
- **计分逻辑**：地主赢（+番数 ×2）、农民赢（每个农民 +番数 ×1）
- **撤销小局**：可撤销最后一条小局记录，分数自动回退
- **大局管理**：支持开始、结束、继续、删除大局
- **历史查看**：弹窗查看大局完整详情的逐局分数变化
- **文件存储**：自动保存到 `games.json`，下次启动自动加载

## 操作说明

| 操作 | 位置 | 说明 |
|------|------|------|
| 修改姓名 | 第一页 | 点击输入框修改，失焦自动保存 |
| 设为地主 | 第一页 | 点击玩家旁的按钮切换 |
| 计分 | 第一页 | 选地主后点"地主赢"/"农民赢" |
| 撤销小局 | 第一页 | 点击"撤销最后小局"按钮 |
| 开始/结束大局 | 第一页 | 下方按钮切换 |
| 继续大局 | 第二页 | 选中已结束的大局 → 点击"继续打" |
| 删除大局 | 第二页 | 选中已结束的大局 → 点击"删除" |
| 查看详情 | 第二页 | 选中大局 → 点击"查看详情" 或直接双击 |
