# 斗地主计分器

> [English](./README.en.md) | 中文

一个基于 Python Tkinter 的斗地主计分图形界面程序，支持大局/小局两层结构、文件持久化、历史管理。

---

## 功能

- 🎴 **三人记分** — 支持修改玩家姓名，每个大局内分数累计，实时显示
- 🔢 **番数调节** — +1 番 / +2 番 / 重置番数
- 👑 **地主标记** — 点击切换地主，同一时间只能有一个地主，按钮显示 👑 标识
- 🎯 **计分逻辑** — 地主赢（+番数 ×2）、农民赢（每个农民 +番数 ×1）
- ↩️ **撤销小局** — 可撤销最后一条小局记录，分数自动回退
- 📊 **大局管理** — 支持开始、结束、继续、删除大局，结束大局后保留玩家名字
- 📋 **历史查看** — 弹窗查看大局完整详情的逐局分数变化
- ⌨️ **快捷键** — 历史大局页面支持 `Enter` 继续打、`Delete`/`Backspace` 删除
- 💾 **文件存储** — 自动保存到 `games.json`，下次启动自动加载

## 环境要求

- Python 3.7+
- tkinter（Python 标准库，无需额外安装）

## 快速开始

```bash
python main.py
```

或双击 `main.py` 文件直接运行。

## 界面截图

### 当前大局页面

<img src="1_当前大局页面.png" width="600">

- 顶部显示大局名称和状态（进行中/未开始）
- 玩家区域：可修改姓名，实时显示累计分数，点击"设为地主"按钮切换地主

<img src="2_历史大局页面.png" width="600">

- 列表展示所有大局（名称、状态、累计分）

## 操作说明

| 操作 | 位置 | 说明 | 快捷键 |
|------|------|------|--------|
| 修改姓名 | 第一页 | 点击输入框修改，失焦自动保存 | — |
| 设为地主 | 第一页 | 点击玩家旁的按钮切换 | — |
| 计分 | 第一页 | 选地主后点"地主赢"/"农民赢" | — |
| 撤销小局 | 第一页 | 点击"撤销最后小局"按钮 | — |
| 开始/结束大局 | 第一页 | 下方按钮切换 | — |
| 继续大局 | 第二页 | 选中已结束的大局 → 点击"继续打" | `Enter` |
| 删除大局 | 第二页 | 选中已结束的大局 → 点击"删除" | `Delete` / `Backspace` |
| 查看详情 | 第二页 | 选中大局 → 点击"查看详情" 或直接双击 | — |
