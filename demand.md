# 斗地主计分器 - Tkinter 实现需求

## 项目目标

使用 Python 的 `tkinter` 库开发一个斗地主计分器图形界面程序，支持三人记分、地主标记、番数调节、胜负计分及操作历史记录。

---

## 功能需求（详细）

### 1. 玩家管理

- 界面显示三个玩家的姓名输入框（默认可填“玩家1”“玩家2”“玩家3”），支持随时修改。
- 每个玩家姓名旁显示其当前分数（整数，初始为 0）。

### 2. 番数控制

- 显示当前番数（初始为 1 番，即“底番”）。
- 提供两个按钮：“+1 番”和“+2 番”，每点击一次对应增加番数（可累计，无上限）。
- 番数可手动重置（建议增加一个“重置番数”按钮，归为 1 番）。

### 3. 地主标记

- 每个玩家姓名旁提供一个“设为地主”按钮。
- 同一时间只能有一个地主，更换地主时自动取消原标记。

### 4. 胜负计分

- **“地主赢”按钮**：当前地主得分增加 `当前番数 × 2`，两位农民分数不变（但可记录信息）。
- **“农民赢”按钮**：两位农民（非地主）各增加 `当前番数 × 1`，地主分数不变。
- 每次点击计分按钮后，**自动记录一条操作历史**（时间戳可选），内容包括：
  - 操作类型（地主赢 / 农民赢）
  - 当前番数
  - 各玩家分数变化量
  - 操作后的各玩家总分
- 历史记录应显示在界面的列表或文本框中（可滚动）。

### 5. 附加功能（建议但非必须）

- 提供“重置全部”按钮，将所有分数归零，番数重置为 1，取消地主标记，清空历史记录。

## 技术实现要求

- 使用 `tkinter` 标准库，无需第三方库。
- 代码结构建议：
  - 使用面向对象方式，主类继承 `tk.Tk` 或 `tk.Frame`。
  - 使用 `StringVar`/`IntVar` 管理姓名和分数，便于界面更新。
  - 历史记录可用 `Listbox` 或 `Text` 组件，并设置 `state='disabled'` 防止编辑，更新时临时开启再关闭。
- 所有按钮响应函数需包含必要的异常处理（如确保已选择地主再计分，否则弹出提示）。

## 输入/输出示例

- 初始：三人分数 0，番数 1，无地主。
- 用户设置玩家1为地主，点击“+2番”两次（番数变为 5），点击“地主赢” → 玩家1分数 +10（5×2），历史记录显示：“地主赢 | 番数:5 | 玩家1 +10 (总分:10)”。
- 若点击“农民赢”，则玩家2和玩家3各 +5（当前番数），玩家1不变。
